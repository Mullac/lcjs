
Jake commands

	jake
	jake test
	jake clean
	jake deploy

GIT commands

	git status
	git add .
	git commit -am "[Commit message]"
	git push

Heroku commands

	heroku create [name of app]
	git push heroku master
	heroku ps:scale web=0   // Stops the app
	heroku ps:scale web=1   // Starts the app

Karma client tests

	karma start
	karma run
